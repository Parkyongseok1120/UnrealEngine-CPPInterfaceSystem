# Unreal Engine 플러그인 개발 가이드

## 1. 플러그인 개념
- **플러그인이란?**  
  플러그인은 Unreal Engine에서 기능을 확장하기 위해 사용하는 모듈입니다.  

- **플러그인의 장점**  
  - 재사용성  
  - 모듈화  
  - 협업 용이  

## 2. 플러그인 구조
- **파일 및 디렉토리 구조**  
  ```  
  MyPlugin/  
  ├── Source/  
  │   ├── MyPlugin/  
  │   ├── MyPluginEditor/  
  ├── MyPlugin.uplugin  
  ```  

- **주요 구성 요소**  
  - `.uplugin` 파일: 플러그인의 메타데이터  
  - Source 디렉토리: 코드 파일  

## 3. 개발 워크플로우
- **플러그인 생성하기**  
  Unreal Engine 에디터에서 "New Plugin" 메뉴를 사용합니다.  
  
- **플러그인 빌드 및 테스트**  
  Visual Studio에서 빌드 후, Unreal Engine 에디터에서 테스트합니다.  

- **플러그인 배포**  
  배포를 위해 플러그인 패키지를 생성하는 방법을 설명합니다.
